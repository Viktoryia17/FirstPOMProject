Структура проекта: 
*config - в этой папке должны быть конфигурации проекта (содержит логин и пароль) 
*fixtures - содержит тестовые данные (нпр. картинки, документы, тектста) 
*pages - содержит Page Object
*test - содержит сценарий
*utils - фунции, которые помогают в различных случаях (нпр. геннерируют пользоа=вателя или товар)
